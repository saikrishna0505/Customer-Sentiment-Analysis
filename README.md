# 📊 Customer Sentiment Analysis using Machine Learning & NLP

## 📌 Overview
This project focuses on analyzing customer reviews and predicting sentiment using **Natural Language Processing (NLP)** and **Machine Learning** techniques. The system classifies customer feedback into **Positive 😊, Negative 😞, or Neutral 😐** categories, helping businesses understand customer opinions and improve decision-making.

The project includes data preprocessing, text vectorization, exploratory data analysis (EDA), model training, evaluation, and sentiment prediction using real-world customer review datasets.

---

## ✨ Features
- 🧹 Text preprocessing and cleaning
- 💬 Sentiment classification (Positive, Negative, Neutral)
- 📈 Exploratory Data Analysis (EDA)
- 📝 TF-IDF and Count Vectorization
- 🤖 Machine Learning model training and evaluation
- 📊 Accuracy and performance visualization
- ⚡ Real-time sentiment prediction capability

